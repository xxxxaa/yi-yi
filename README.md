# YiYi（易易）

房产中介智能助手 - 让每一次推荐都更专业

## 项目简介

YiYi 是一款面向房产中介从业者的 AI 智能助手，帮助经纪人：

- **管理楼盘物料** - 上传、解析、结构化存储各类楼盘资料
- **生成营销内容** - 一键生成小红书推文、短视频脚本、朋友圈文案
- **智能匹配客户** - 基于客户需求自动推荐合适房源

## 核心功能

### 📁 楼盘物料管理

- 支持 PDF、图片、视频、Excel 等多种格式
- AI 自动解析提取关键信息
- 结构化存储，随时检索

### ✍️ 内容生成引擎

- 小红书种草笔记
- 抖音/视频号短视频脚本
- 朋友圈推广文案
- 多种风格模板可选

### 👥 客户管理

- 客户需求画像
- 房源智能匹配
- 跟进提醒
